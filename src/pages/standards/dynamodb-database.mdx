# DynamoDB Database

Most of Topcoder intenal data is stored on Amazon DynamoDB.
We recommends using Amazon DynamoDB as its about fully managed proprietary NoSQL database service that supports key-value and document data structures and is offered by Amazon.com as part of the Amazon Web Services portfolio.
